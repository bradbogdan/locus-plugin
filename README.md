# Locus plugin marketplace

Private marketplace hosting the **Locus** Cowork/Claude Code plugin, a private chief-of-staff workspace (triage, daily briefing, weekly status, knowledge pill), with the full Locus branding.

## Install
- **Cowork desktop**: Directory to Plugins to `+` to Add from a repository, paste this repo's URL.
- **Claude Code CLI**: `claude plugin marketplace add bradbogdan/locus-plugin` then `claude plugin install locus@locus`.

Private repo: contains Majoritas brand assets. Do not make public without Majoritas sign-off.
